# CSM-ModSets-TagRouter

基于 [CSM（可通信状态机）](https://nevstop-lab.github.io/CSM-Wiki/) 框架的波形到 TagDB 数据路由模块，负责将波形数据转换为 TagDB 标签数据。

This repository contains a CSM-based module for routing waveform data to TagDB tags.

---

## 模块列表

| 模块 | 所属模块组 | 功能说明 |
| --- | --- | --- |
| [TagRouter](./TagRouter.md) | CSM-TagRouter.lvlib | 波形转 TagDB，提取波形属性并写入标签数据库 |
